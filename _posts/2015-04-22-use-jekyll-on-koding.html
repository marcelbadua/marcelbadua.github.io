Jekyll is a simple, blog-aware static site generator that is supported by GitHub Pages. It ables you to integrate common header and footers for all of your static pages.

If you are using Koding for your projects, you might want to install Jekyll to preview you blog site. Open up your terminal and run the following lines.

gpg --keyserver hkp://keys.gnupg.net --recv-keys D39DC0E3
\curl -sSL https://get.rvm.io | bash -s stable
source ~/.rvm/scripts/rvm
rvm install ruby-2.2.0
gem install jekyll
jekyll new jekyllsite
cd jekyllsite
jekyll serve -d /home/ {username} /Web
